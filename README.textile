h2. Specify Access key and Filmtipset user ID

# *Rename <code>access_key.py.sample</code>* to <code>access_key.py</code>
# Add a proper *access key* and *user id* in access_key.py. Get your access key by following the instructions on "Filmtipset's API page":http://nyheter24.se/filmtipset/api.cgi, and get your user id by "searching for your username":http://nyheter24.se/filmtipset/search_member.cgi and mouseover your nick, the member parameter is your user id. With those two changes, all requests are identified as your own (so I don't get the blame if you spam the API), and all grades will be based on your own accout.
